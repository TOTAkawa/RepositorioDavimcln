Machine Learning e Deep Learning

Machine Learning

Machine Learning é uma área da inteligência artificial que permite que sistemas aprendam com dados e façam previsões ou decisões sem serem explicitamente programados.

Deep Learning

Deep Learning é uma subárea do Machine Learning que usa redes neurais artificiais profundas para aprender representações complexas dos dados.
